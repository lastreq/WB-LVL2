Ускоряем работу Windows XP

6 декабря, 2012, Категория: Cтатьи про it

Как известно, операционная система Windows XP вышла в 2001 году, но до сих пор остается очень популярной. Несмотря на то, что вышло несколько более новых версий Windows, Windows XP продолжают ставить либо по привычке, либо на старые компьютеры, которые не способны работать с более требовательными операционными системами.

Посмотрев системные требования Windows XP, можно убедиться, что это отличный вариант для слабых машин даже в наше время: Минимальные требования к оборудованию для установки Windows XP:

  • Процессор Pentium с частотой 233 МГц или более быстрый (рекомендуется не менее 300 МГц)
  • Не менее 64 МБ оперативной памяти (рекомендуется не менее 128 МБ)
  • Не менее 1,5 ГБ свободного места на жестком диске
  • Дисковод для компакт- или DVD-дисков
  • Клавиатура, мышь Microsoft Mouse или совместимое указывающее устройство
  • Видеокарта и монитор, поддерживающие режим Super VGA с разрешением не менее чем 800x600 точек
  • Звуковая плата Динамики или наушники

Как правило, после установки Windows XP, опытный пользователь всегда выполняет некоторые несложные действия, чтобы ускорить и оптимизировать работу системы. В статье я приведу несколько советов и твиков Windows XP, которые помогут сделать ее еще более быстрой.

Перед тем, как перейти к советам по оптимизации Windows XP, напишу пару рекомендаций по установке системы:

1. Устанавливать любую версию Windows лучше через BIOS, таким образом, можно избежать многих проблем и ошибок при установке и работе установленной ОС. Если устанавливать новую Windows путем обновления старой, это может не дать никакого эффекта, и все проблемы, вирусы и ошибки останутся.

2. При переустановке Windows желательно всегда делать форматирование. Не обязательно форматировать весь жесткий диск, достаточно это сделать только с тем его разделом, в котором была установлена прежняя ОС.

3. Для Windows лучше создать отдельный раздел на жестком диске и оставить его для ОС и устанавливаемых программ. А свои файлы хранить в другом разделе, тогда при переустановке системы ваши данные не пострадают. Для Windows XP рекомендуется делать системный раздел размером от 15Гб и более, но определиться с размером системного диска проще, зная, какие программы будут устанавливаться на компьютер, и для чего он будет использоваться. Поэтому оптимальный размер системного раздела проще определить на практике, уже пользуясь компьютером.

После установки ОС вы видите чистый монитор и всеми любимые обои с голубым небом и зеленой травой. Что же сделать, чтобы настроить и оптимизировать систему? Итак, начнем:

1. Самое первое и простое - отключить лишние визуальные эффекты. На слабых компах сразу видно, как все начинает тормозить, пытаясь воспроизвести тяжелую анимацию сворачивания окон, плавного появления меню и т.д. Чтобы отключить визуальные эффекты в Windows XP делаем следующее - щелкнуть правой кнопкой по ярлыку Мой компьютер на рабочем столе, в появившемся контекстном меню выбираем пункт Свойства. В открывшемся окне переходим на вкладку Дополнительно, видим надпись Быстродействие, щелкаем рядом с ней кнопку Параметры. Откроется окно Параметры быстродействия, в котором на вкладке Визуальные эффекты настраиваем эффекты на свое усмотрение. Для достижения максимального быстродействия стоит отключить практически все эффекты, кроме Использования типичных задач для папок и использования тем оформления. Можно отключить и их, но это снизит удобство пользования системой.

2. Возможно, время появления выпадающих меню может показаться вам слишком долгим, это легко исправить в реестре. Настраивается с помощью реестра. Вызываем командную строку сочетанием клавиш win(флажок)+r либо из меню Пуск (Пуск -> Выполнить ). В командной строке пишем regedit, жмем Enter, появится окно редактора реестра. В редакторе реестра переходим к разделу HKEY_CURRENT_USER\Control Panel\Desktop, находим там параметр MenuShowDelay, по умолчанию у него стоит значение 400, для ускорения выпадения меню лучше всего поставить значение 100-200. Перезагружаем систему для применения изменений.

3. По умолчанию в Windows XP запущены некоторые лишние службы, которые используют память и могут тормозить работу компьютера. Отключаем их. Настройка служб производится в Панели управления: Панель управления -> Администрирование -> Службы. В появившемся окне увидим таблицу с названием службы, ее состоянием - работает она в текущий момент или нет и типом ее запуска - автоматически, вручную или отложенный запуск.

Как правило на Windows XP отключаются следующие службы:

- Автоматическое обновление (обновления могут тормозить работу старых компьютеров, вызывать ошибки и лаги системы, а после их установки, обычно компьютер требует перезагрузки не в самый подходящий момент)

- Брандмауэр Windows (большинство пользователей все равно устанавливают сторонние антивирус и файрвол, таким образом необходимости во встроенном брандмауэре нет, тем более, работает он не лучшим образом)

- Вторичный вход в систему и Совместимость быстрого переключения пользователей следует отключить, если на компьютере только один пользователь, либо если несколько, но данные возможности не нужны и не используются

- Планировщик заданий - отключить, если не пользуетесь

- Беспроводная настройка - если у вас нет подключения к сети Wi Fi отключаем

- Служба времени Windows проводит синхронизацию времени с атомными часами в интернете. Отключите, если в этом нет необходимости

- Служба COM записи компакт-дисков - отключить, если не используете встроенные средства записи дисков windows. Обычно их никто не использует, а пользуется для этих целей программами вроде Nero Burning ROM, поэтому можно смело отключать

- Служба индексирования - ее отключение ускоряет работу системы и открытия файлов, но замедляет поиск файлов.

Помните, что отключение служб влияет на работу системы и при неправильном использовании может привести к ошибкам и падениям. Поэтому будьте внимательны и осторожны.

4. Увеличиваем файл подкачки. Файл подкачки используется в случае, когда размеров физической оперативной памяти недостаточно. Можно самостоятельно настроить размер файла подкачки:

Щелкнуть правой кнопкой по ярлыку Мой компьютер на рабочем столе, в появившемся контекстном меню выбираем пункт Свойства. В открывшемся окне переходим на вкладку Дополнительно, видим надпись Быстродействие, щелкаем рядом с ней кнопку Параметры. Откроется окно Параметры быстродействия, в котором переходим на вкладку Дополнительно. На данной вкладке обращаем внимание, чтобы под надписью "Оптимизировать работу" переключатель был установлен напротив варианта программ. Если все в порядке, щелкаем кнопку Изменить. В открывшемся окне с заголовком Виртуальная память можно изменять размер файла подкачки. Если не знаете, какой размер выбрать, выберите параметр Размер по выбору системы. Обычно размер файла подкачки зависит от объема оперативной памяти и равен ему. Например, если у вас 1гб оперативной памяти, то файл подкачки должен быть установлен 1гб, а в поле максимальный размер пишем значение в два раза больше - 2гб. После применения изменений перезагружаем компьютер.

5. Отключаем автозапуск дисков и подключаемых носителей. Редактируем реестр:

По пути HKEY_LOCAL_MACHINE\SOFTWARE\Microsoft\Windows\CurrentVersion\Policies создаем новую папку (раздел) Explorer, в нем создаем параметр NoDriveTypeAutoRun. В параметре можно задать следующие шестнадцатеричные значения: 0x1 - отключить автозапуск приводов неизвестных типов 0x4 - отключить автозапуск съемных устройств 0x8 - отключить автозапуск интегрированных устройств 0x10 - отключить автозапуск сетевых дисков 0x20 - отключить автозапуск CD-приводов 0x40 - отключить автозапуск RAM-дисков 0x80 - отключить автозапуск на приводах неизвестных типов 0xFF - отключить автозапуск вообще всех дисков. Также есть другой способ: Пуск -> Выполнить -> пишем в командной строке gpedit.msc -> Конфигурация компьютера -> Административные шаблоны -> Система -> Отключить автозапуск Перезагружаем компьютер для применения изменений.

6. Включаем клавишу Num Lock при загрузке. Запускаем редактор реестра. В ветке HKEY_CURRENT_USER\Control Panel\Keyboard меняем значение InitialKeyboardIndicators с 0 на 2 Перезагружаем компьютер

7. Отключаем поиск сетевых компьютеров при запуске проводника. Данная функция может тормозить открытие папок, так как пытается искать сетевые компьютеры и принтеры даже если их нет. В меню Пуск переходим в Панель управления, щелкаем Оформление и темы, выбираем Свойства папки, на вкладке Вид снимаем флажок с параметра Автоматический поиск сетевых папок и принтеров. Также окно Свойства папки может быть вызвано из меню любой папки.

8. Отключаем поддержку Zip архивов. Функция, которая позволяет Windows воспринимать и открывать zip архивы как обычные папки, на низко производительных машинах может вызвать тормоза. В командной строке выполняем следующую команду: regsvr32 /u zipfldr.dll Чтобы вернуть поддержку ZIP-файлов: regsvr32 zipfldr.dll

9. Делаем дефрагментацию. Периодически необходимо делать дефрагментацию, чтобы упорядочить содержимое локальных дисков и оптимизировать работу системы. В ОС Windows есть встроенная утилита для дефрагментации, вызывается она либо из Панели управления: Пуск –> Настройка –> Панель управления –> Администрирование –> Управление компьютером –> Дефрагментация диска –> выделите нужный диск –> Дефрагментация Либо из контекстного меню диска: Свойства -> Сервис -> Дефрагментация диска -> Выполнить дефрагментацию Для того, чтобы понять, нужна дефрагментация или нет, достаточно выделить нужный диск и нажать кнопку Анализ. Спустя некоторое время система выдаст результат - нужна дефрагментация или нет. Если дефрагментация нужна, нажмите кнопку Дефрагментация. На больших объемах диска и медленном компьютере дефрагментация может занять продолжительное время. Рекомендуется проводить дефрагментацию раз в 1-2 месяца.

Используя приведенные выше советы, вы сможете ускорить работу своей системы, а также продлить срок ее службы без переустановки. Данная статья не является исчерпывающей и, при желании, вы найдете в сети еще много способов ускорить и оптимизировать свою Windows XP.
